# ربات دعوت‌نامه تلگرام

این ربات تلگرام برای ایجاد دعوت‌نامه‌های شخصی‌سازی شده استفاده می‌شود.

## نصب و راه‌اندازی

1. نصب کتابخانه‌های مورد نیاز:
```bash
pip install -r requirements.txt
```

2. دریافت توکن ربات از @BotFather در تلگرام

3. جایگزین کردن `YOUR_BOT_TOKEN_HERE` در فایل `bot.py` با توکن واقعی

4. قرار دادن تصاویر دعوت‌نامه:
   - `invitation_male.jpg` - دعوت‌نامه مخصوص آقایان
   - `invitation_female.jpg` - دعوت‌نامه مخصوص خانم‌ها

5. (اختیاری) قرار دادن فونت فارسی در پوشه `fonts/`:
   - `Vazir.ttf` یا `Shabnam.ttf`

6. اجرای ربات:
```bash
python bot.py
```

## استفاده

1. شروع ربات با دستور `/start`
2. انتخاب جنسیت (آقا یا خانم)
3. وارد کردن نام و نام خانوادگی
4. دریافت تصویر دعوت‌نامه شخصی‌سازی شده
